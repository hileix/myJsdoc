# JSDoc Template

JsDoc 文档模板

## 配置项

```json
{
    "templates": {
      "name": "Doc Template", // 文档名称
      "tabNames":{
          "tutorials":"wiki", // 导航名称
          "apiName":"api" 
      }
  }
}
```

### 如何自定义

- tmpl 目录 `layout.tmpl ` 修改布局
- `publish.js`
